About us

We are a group of six NTU Students working on AQUA SAVER, a solution that could potentially save drowning individuals while swimming.

We started this project for a module called "EE5083 - From Passion to Action: Six Key Steps to a Start-up!"

How it works

We will be partnering with Condomoniums on a subscription-based model. Aqua Saver will then be connected to the CCTV camera of the Condomoniums's Swimming Pools.


An AI-driven video analytic tool which uses 3D Convulational Neural Network , will run on the live stream data of the connected camera to detect potential drowning cases and then alert the lifeguard nearby.
